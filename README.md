This is a fun, interactive console-based Rock, Paper, Scissors game built in Java.
The player competes against the computer, which makes a random choice each round.

The game continues until the player chooses to stop.

FEATURES

Player can choose:

Rock

Paper

Scissors

Computer makes a random choice

Automatically determines:

Win

Lose

Tie

Option to play again

Handles invalid input

Beginner-friendly and fun to use

TECHNOLOGIES USED

Java

Scanner class for user input

Random class for computer selection

HOW TO RUN THE PROGRAM

Save the file as:
RockPaperScissors.java

Open a terminal or command prompt in the project folder

Compile the program:
javac RockPaperScissors.java

Run the program:
java RockPaperScissors

HOW THE GAME WORKS

The user enters: Rock, Paper, or Scissors

The computer randomly selects one of the three

The program compares both choices

The winner is displayed

The player is asked if they want to play again

GAME RULES

Rock beats Scissors

Scissors beats Paper

Paper beats Rock

Same choice = Tie
